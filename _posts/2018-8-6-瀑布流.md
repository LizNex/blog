## flex 瀑布流

使用flex完成瀑布流主要的flex属性 `flex-direction`

外层横向，内部列纵向就好了

```css
    .container {
        display: flex;
      }
      .col {
        display: flex;
        flex-direction: column;
        flex: 1;
      }
      .item {
        box-sizing: border-box;
        padding: 4px;
        width: 100%;
      }
      .item__img {
        width: 100%;
```
